# 化学方程式互动实验室

一个面向课堂演示的单页 Web 应用，让化学元素的"相遇"过程直观、可交互、易复用。

## 功能特性

### 元素周期表互动

- 点击任意元素，系统自动高亮可与之发生反应的其他元素
- 不可反应的元素会变暗，引导用户快速找到有效组合
- 支持元素搜索定位，输入符号或名称即可跳转

### 反应方程式生成

- 选择元素组合后，实时展示配平后的化学方程式
- 标注反应类型（化合、分解、置换、复分解等）和反应条件
- 内置 400+ 条化学反应数据，覆盖中学化学常见反应

### AI 智能解释

- 点击反应可唤起 AI 解释弹窗，流式输出详细解读
- 内容涵盖反应原理、工业应用、安全提示、延伸知识
- 帮助学生理解反应背后的逻辑，而非死记方程式

### 双模式探索

- **元素高亮模式**：从元素周期表出发，探索元素间的反应关系
- **化合物链式模式**：从化合物出发，追溯其前驱反应物，层层展开

### 反应网络图

- 以可视化图谱展示化合物的前驱反应关系
- 点击化合物节点可展开其前驱反应物和反应条件
- 同一化合物在整张图中只允许展开一次，避免重复和跨层级连线
- 再次点击已展开节点可收拢其前驱结构

### 收藏与本地存储

- 一键保存常用方程式到收藏夹
- 数据存储在浏览器本地，下次打开无需重新查找
- 收藏夹支持快速查看和删除管理

### 响应式设计

- 适配桌面端鼠标操作和移动端触摸操作
- 布局自动适应不同屏幕尺寸
